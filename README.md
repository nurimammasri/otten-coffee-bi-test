# Otten Coffee - BI Engineer Technical Test

This repository contains my solutions for the **Otten Coffee BI Engineer Technical Test**.

## Repository Structure

The test is divided into three main sections:

### 1. Test 1 - Advanced SQL
Contains complex SQL queries answering various analytical business questions.
- `Test 1/test1_advanced_sql.sql`: SQL scripts for the queries.
- `Test 1/test1_explanation.html`: Detailed explanations and results of the queries.

### 2. Test 2 - Data Transformation & Analytics
Focuses on data processing, aggregation, and generating insights from raw data.
- `Test 2/test2_explanation.html`: Detailed explanation of the analytics workflow.

### 3. Test 3 - Data Pipeline & Architecture
Demonstrates a structured data warehouse pipeline approach (Staging -> Intermediate -> Mart).
- `Test 3/01_staging/`: Staging models for raw data ingestion.
- `Test 3/02_intermediate/`: Intermediate models for joining and transforming data.
- `Test 3/03_mart/`: Mart models containing final business logic and aggregations.
- `Test 3/test3_explanation.html`: Documentation of the pipeline architecture.
- `Test 3/test3_pipeline.zip`: Additional pipeline assets.

## About
This project was developed as part of the recruitment process for the Business Intelligence Engineer position at Otten Coffee.
